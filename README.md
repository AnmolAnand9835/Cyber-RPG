# ⚡ Cyber RPG

A cyberpunk-themed text RPG Discord bot built with Node.js, Discord.js, and MongoDB.

## 🚀 Overview

Cyber RPG lets players create their own digital operative and progress through a fictional cyberpunk world by earning credits, leveling up, collecting equipment, and completing missions.

All actions and missions are fictional game mechanics designed for entertainment and role-playing.

## ✨ Planned Features

* 👤 Player Profiles
* 💰 Credits & Economy System
* 📈 XP & Leveling
* 🎒 Inventory & Equipment
* 🎁 Daily Rewards
* 🛒 Shop System
* 🎯 Missions & Quests
* 👾 Boss Raids
* 🏆 Leaderboards
* 🤝 Trading System
* ⚔ PvP Battles
* 🌍 Global Events

## 🛠 Tech Stack

* Node.js
* Discord.js
* MongoDB
* Mongoose

## 📦 Installation

```bash
git clone https://github.com/AnmolAnand9835/Cyber-RPG.git
cd Cyber-RPG
npm install
```

Create a `.env` file:

```env
TOKEN=your_discord_bot_token
MONGODB_URI=your_mongodb_connection_string
```

Run the bot:

```bash
node index.js
```

## 🎮 Vision

Cyber RPG aims to become a fun, community-driven Discord RPG where players can build their operatives, explore an evolving digital universe, and compete with friends.
